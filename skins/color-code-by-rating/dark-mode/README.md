# In order to use the dark mode skin...
1. On your AO3 sidebar, click on "Skins" (it should be under "Preferences" in the same section as "Dashboard")
2. Click "Create Site Skin" (button should be on the top right)
3. In the "Title" section, give the skin a UNIQUE name (I recommend "[yourusename].color code by rating")
4. Scroll down to the CSS textbox and paste the contents of **code.css**
5. Scroll down farther to "Advanced" and click the "Show" button
6. Scroll down to "Parent Skins" and click "Add parent skin"
7. Input "Reversi" as the Parent Skin
8. Scroll down and click "Submit"
9. Scroll down and click "Use"

# This skin should...
- Change a fic's blurb color in your Works/Bookmarks/Search based on the fic's rating (General = green, Teen = yellow, Mature = orange, Explicit = red, Not Rated = black/gray)
- When hovering over a link in a fic's blurb (including tags), the link will be highlighted appropriately
- Buttons have been changed to match the color scheme
- When reading a fic, these changes should persist in the fic's card at the top and the buttons at the top and bottom of the fic (not including the Kudos list and the comment area)
- Any links in the fic should behave similarly to tags: they will highlight appropriately based on the fic's rating
- WHEN ON A COMPUTER: Highlighting text inside a fic's blurb or inside the fic itself should display an appropriate color

*If any of these things aren't working properly, [please let me know](https://sssrha.tumblr.com/ask)!!*
